# MathStudyApp

大学数学を問題演習形式で学ぶためのWebアプリです。

## 現在の機能

- 線形代数、微積分、抽象代数、確率統計学の問題演習
- セクション別の体系的な学習順
- セクションごとの習得率表示
- 難易度フィルター
- 計算問題と概念問題のタイプ別フィルター
- MathJaxによるTeX形式の数式表示
- 解答履歴のローカル保存
- 間違えた問題の復習リスト
- 正答率、演習数、連続正解数の表示

## 線形代数の範囲

- ベクトル空間と一次独立
- 行列と基本計算
- 連立一次方程式
- 線形写像
- 固有値と対角化
- 内積と直交性
- 対称行列と二次形式
- 標準形と行列分解
- 基本部分空間と応用的分解

## 抽象代数の範囲

- 群と部分群
- 共役類、中心、類方程式、Sylow 定理
- 群準同型と商群
- 同型定理、自己同型、単純群
- 環とイデアル
- 素イデアル、極大イデアル、中国剰余定理
- 多項式環、PID、UFD、既約判定
- 体、有限体、代数拡大、超越元
- 加群、商加群、完全列、群作用
- 正規拡大、分離拡大、ガロア理論の入口

## 確率統計学の範囲

- 確率の基本
- 確率変数と期待値
- 代表的な確率分布
- 記述統計
- 推定
- 仮説検定
- 回帰と相関

## 使い方

`index.html` をブラウザで開くと利用できます。学習記録はブラウザの `localStorage` に保存されます。

## 今後の拡張

`app.js` の `subjects` 配列に分野と問題を追加すると、微積分、抽象代数、確率統計学などの分野も扱えるようになります。
