# MathStudyApp

大学数学を問題演習形式で学ぶためのWebアプリです。

## 現在の機能

- 線形代数の問題演習
- セクション別の体系的な学習順
- セクションごとの習得率表示
- 難易度フィルター
- 解答履歴のローカル保存
- 間違えた問題の復習リスト
- 正答率、演習数、連続正解数の表示

## 線形代数の範囲

- ベクトル空間と一次独立
- 行列と基本計算
- 連立一次方程式
- 線形写像
- 固有値と対角化
- 内積と直交性
- 対称行列と二次形式
- 標準形と行列分解
- 基本部分空間と応用的分解

## 使い方

`index.html` をブラウザで開くと利用できます。学習記録はブラウザの `localStorage` に保存されます。

## 今後の拡張

`app.js` の `subjects` 配列に分野と問題を追加すると、微積分や抽象代数などの分野も扱えるようになります。
